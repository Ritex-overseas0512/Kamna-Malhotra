# Kamna Malhotra Digital vCard

Premium digital business card for Kamna Malhotra, GM- Sales and Marketing at Ritex Overseas Pvt Ltd.

## Files

- `index.html` — main card page
- `styles.css` — premium Apple-style card styling
- `kamna-malhotra.vcf` — downloadable contact card
- `assets/ritexlogo.svg` — Ritex logo
- `assets/profile.png` — placeholder profile image
- `vercel.json` — Vercel config

## Important

Replace `assets/profile.png` with Kamna Malhotra's real profile photo using the same file name.

## Deploy on Vercel

Upload this folder to GitHub and import it into Vercel. Use framework preset `Other`.
